# Fm2003 Tap

## How do I install these formulae?

`brew install fm2003/tap/<formula>`

Or `brew tap fm2003/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "fm2003/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
